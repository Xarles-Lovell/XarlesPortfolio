## 项目是什么

这是 Xarles 的个人作品集网站，整体包装成一份"犯罪档案"——Xarles 本人是"嫌疑人"，设计作品是"案件记录"，访客（HR）是来调查的侦探。

**核心叙事逻辑：**
- 身份包装：不写"关于我"，写"S级悬赏令 / 嫌疑人档案"
- 能力包装：不写"擅长平面和建筑设计"，写"伪装能力 / 作案年限"
- 作品包装：不写"项目展示"，写"犯罪现场记录 / 案件卷宗"
- 联系包装：不写"联系我"，写报纸头条"嫌疑人主动现身"

**平衡原则：外壳是戏，内容是真。** 包装语言用于吸引注意力，点进案例详情后是完整专业的设计项目介绍。

## 视觉风格

### 色彩系统
| 变量 | 色值 | 用途 |
|------|------|------|
| `--bg` | `#A9856F` | 页面背景（暖棕色） |
| `--paper` | `#F4E8DC` | 主卡片/卷宗纸（粉米白） |
| `--paper-alt` | `#EAD8C4` | 次级纸张 |
| `--ink` | `#261717` | 主文字（深褐墨色） |
| `--ink-light` | `#6b4040` | 次级文字 |
| `--red` | `#8b1a1a` | 点缀色（暗红，用于强调） |
| `--tape` | `rgba(175,140,118,0.52)` | 胶条色 |

堆叠底纸：`#e0c9b4`（旋转 +2.4°）、`#ead5bc`（旋转 -1.8°）

### 字体
- **标题**：Playfair Display（衬线，900 weight）
- **英文 label / 按钮**：Special Elite（打字机感）
- **正文 / 中文**：Noto Serif SC

### 核心设计原则
- 棕色背景 + 米白卷宗纸，形成强对比
- 纸张质感：四角磨损渐变 + SVG 纤维纹理
- 堆叠感：主纸下方两张底纸，+2.4° / -1.8° 偏移
- **胶条规则**：卷宗本身不贴胶带；卷宗内的照片才贴胶条。大图左右各一条（±10°），小图顶部居中一条
- 印章："已结案"文字边框样式，旋转约 -4°

## 各页面关键细节

### 首页（index.html）
- 通缉令卡片：`max-width: 680px`，旋转 -0.4°，两层底纸堆叠
- 危险指数：4个实心红点 + 1个空心点
- 档案信息行：代号、状态、伪装能力、作案年限、已知罪行、悬赏
- 嫌疑人陈述：中文简介，待填入真实内容
- 按钮：深色填充，悬停时红色从左滑入，链接到 cases.html

### 案例列表页（cases.html）
- 白板：`max-width: 1080px`，四角胶带固定，轻微纸张纹理
- 5张卡片**宝丽来风格**：白色底板（padding 8px）+ 正方形图片 + 下方白色文字区
- 卡片分布：横向排列，01/03/05 偏上，02/04 偏下，蛇形流动
- 只有旋转角度不规则（-3° 到 +4°），位置整体有序
- 箭头：01→02→03→04→05，JS 动态绘制贝塞尔曲线，交替弯曲方向
- 旁注：卡片正下方手写备注，opacity 0.3
- **右下角固定"新消息"气泡**：红色脉冲圆点 + "收到新消息 · 嫌疑人主动现身——"，链接到 contact.html，页面加载 0.8s 后滑入

### 案例详情页（case-01 ～ case-05）
- Section 结构：犯罪现场 → 案件分析 → 作案手法 → 物证清单 → 案件结论
- 左侧红色虚线装订线贯穿全文
- 主纸下方两张底纸堆叠（同首页逻辑）
- 照片胶条：大图左右各一条（±10°），小图顶部居中
- 底部导航：上一案件 / 返回列表 / 下一案件
- **翻纸动效**：绕边缘旋转翻出，新页从对侧翻入+回弹。方向通过 sessionStorage 传递，需真实浏览器才完整生效
- 每页 `currentNum` 变量对应案件编号（1-5）

### 联系页（contact.html）
- **形式：报纸风格**，刊名"Design Criminal Gazette"
- 第三方口吻，完全规避求职者姿态
- 主标题：「S级在逃嫌疑人主动现身——是陷阱，还是金盆洗手？」
- 两栏布局：左栏事件报道 + 引语框；右栏联系渠道公告 + 案情摘要
- 首字下沉，双线报头，四角方块装饰的联系方式框

## 技术栈 & 部署

- **纯 HTML / CSS / JavaScript**，无框架依赖
- 字体通过 Google Fonts CDN 加载
- **托管**：Netlify（免费）
- **图片**：Cloudinary 存储，通过 URL 引用
- **版本控制**：GitHub（私密仓库）
- **防搜索引擎收录**：根目录放 `_headers` 文件，内容：
  ```
  /*
    X-Robots-Tag: noindex
  ```

## 待填写的占位内容

所有 `[ 方括号 ]` 内容需替换为真实信息：

- `index.html`：昵称、嫌疑人陈述简介
- `case-XX.html`：案件名称、设计类型、完成年份、委托方、一句话描述、案件分析正文、作案手法步骤、物证清单、案件结论
- `contact.html`：邮箱、微信号、其他平台信息

**图片替换方式：** 把占位格
```html
<div class="photo-placeholder main-ph">...</div>
```
换成
```html
<img src="你的Cloudinary图片URL" alt="描述" style="width:100%;height:100%;object-fit:cover;">
```

## 已确认的设计决策（勿改）

- 整体深色棕色背景，不改为浅色方案
- 卡片旋转是唯一的不规则变量，其他元素保持有序
- 卷宗本身不贴胶带，只有卷宗内的照片用胶条
- 箭头连线按 01→02→03→04→05 顺序
- 联系页用第三方报纸口吻，不用第一人称求职语气
- 联系页入口通过 cases.html 右下角"新消息"气泡触达
