## 项目是什么

这是 Xarles 的个人作品集网站，整体包装成一份"案件档案"——Xarles 本人是"嫌疑人"，设计作品是"案件记录"，访客（HR）是来调查的侦探。

**核心叙事逻辑：**
- 身份包装：不写"关于我"，写"S级悬赏令 / 嫌疑人档案"
- 能力包装：不写"擅长平面和建筑设计"，写"伪装能力 / 作案年限"
- 作品包装：不写"项目展示"，写"作案现场记录 / 案件卷宗"
- 联系包装：不写"联系我"，写报纸头条"嫌疑人主动现身"

**平衡原则：外壳是戏，内容是真。** 包装语言用于吸引注意力，点进案例详情后是完整专业的设计项目介绍。

## 视觉风格

### 色彩系统
| 变量 | 色值 | 用途 |
|------|------|------|
| `--bg` | `#7A5A44` | 页面背景（深暖棕色） |
| `--paper` | `#F4E8DC` | 主卡片/卷宗纸（粉米白） |
| `--paper-alt` | `#EAD8C4` | 次级纸张 |
| `--ink` | `#1E1010` | 主文字（深褐墨色） |
| `--ink-light` | `#5a3030` | 次级文字 |
| `--red` | `#A01E1E` | 点缀色（暗红，用于强调） |
| `--tape` | `rgba(175,140,118,0.52)` | 胶条色 |

堆叠底纸：`#e0c9b4`（旋转 +2.4°）、`#ead5bc`（旋转 -1.8°）

### 字体
- **标题**：Playfair Display（衬线，900 weight）+ Noto Serif SC 900
- **英文 label / 按钮**：Special Elite（打字机感）
- **正文 / 中文**：Noto Serif SC

### 核心设计原则
- 深棕色背景 + 米白卷宗纸，形成强对比
- 纸张质感：四角磨损渐变 + SVG 纤维纹理
- 堆叠感：主纸下方两张底纸，+2.4° / -1.8° 偏移
- **胶条规则**：卷宗本身不贴胶带；卷宗内的照片才贴胶条
- 印章：使用 `已结案.png` 图片，顶部顺时针 3°，底部逆时针 8°

## 各页面关键细节

### 首页（index.html）
- 通缉令卡片：`max-width: 780px`，旋转 -0.4°，两层底纸堆叠
- 不居中，从顶部开始（需要滚动才能看完）
- 按钮：深色填充，悬停时红色从左滑入

### 案例列表页（cases.html）
- 白板：`max-width: 1280px`，四角胶带固定
- 5张卡片宝丽来风格：200px 宽
- 卡片分布：横向排列，01/03/05 偏上，02/04 偏下
- 右下角消息气泡：浅色底，hover 变深色红边

### 案例详情页（case-01 ～ case-05）
- 卷宗 `max-width: 1020px`，body padding 12px
- **顶部导航栏**：三栏布局（返回 | 进度轴居中 | 空占位），`width: 100vw`，padding 28px
- **进度轴**：`case01 —— case02 —— case03 —— case04 —— case05`，当前页高亮，可点击跳转
- **返回按钮**：18px，font-weight 700，文字"返回"
- Section 结构：作案现场 → 案件分析 → 作案手法 → 物证清单 → 案件结论
- 左侧红色虚线装订线贯穿全文
- 分割线：只保留 header 下方和导航上方两条（浅色 1.5px），section 之间无分割线
- section 标题 h2：24px/900，红色 span 也是 900
- 已结案印章：顶部 80px 顺时针 3°，底部 56px 逆时针 8° 靠右
- **底部导航**：深色按钮样式（14px/700），hover 红色滑入
  - case-01：← 所有案件 / 下一案 →
  - case-02~04：← 上一案 / 下一案 →
  - case-05：← 上一案 / 所有案件 →
- **页面切换动画**：淡出+上移（只动最上面那张纸，底纸不动）
- **右下角消息气泡**：浅色底，红点脉冲，hover 变深色；"收到新消息" + "嫌疑人主动现身——"

### 联系页（contact.html）
- **背景层：报纸风格**，刊名"Design Criminal Gazette"，两栏布局
- **前景层：Galgame 对话框**，固定在页面底部
- 对话角色：嫌疑人视角，4 段对话逐字打出
- 按钮逻辑：第一段只有"继续"，第二段开始左"返回"右"继续"，最后一段点继续=关闭对话框
- × 关闭按钮：隐藏整个 overlay（opacity 0 + pointer-events none）
- 关闭后右下角显示"回顾对话"按钮
- 左上角返回：根据 sessionStorage fromCase 动态指向来源页

## 已确认的设计决策（勿改）

- 整体深色棕色背景，不改为浅色方案
- 卡片旋转是唯一的不规则变量，其他元素保持有序
- 卷宗本身不贴胶带，只有卷宗内的照片用胶条
- 箭头连线按 01→02→03→04→05 顺序
- 联系页 Galgame 对话框用嫌疑人视角
- 联系页入口通过每个 case 页和 cases 页右下角的消息气泡触达
- 消息气泡默认浅色底，hover 变深色红边
- 进度轴可点击，格式 case01 —— case02 ...
- 主标题字重统一 900（含 Noto Serif SC 900），红色 span 同样 900
- 全站最小字号不低于 11px
